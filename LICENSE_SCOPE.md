# 许可证适用范围

仓库根目录的 `LICENSE` 只覆盖发布者确实拥有著作权或有权再许可的原创代码、文档与模板修改，不会自动改变第三方文件、商标、网站 Logo、产品图片或易优 CMS 本体的许可证。

| 路径/内容 | 许可状态 | 公开发布前要求 |
| --- | --- | --- |
| `scripts/`、`docs/`、本仓库原创模板修改 | 计划按 MIT 发布 | 发布者确认这些内容确为原创或已取得再许可权 |
| 易优 CMS 程序本体 | 不在仓库内，也不受 MIT 覆盖 | 使用者自行合法取得并遵守易优 CMS 条款 |
| PhotoSwipe 5.4.4 | MIT；上游许可证随包保留 | 发布时同时保留 `template/pc/skin/Lib/photoswipe/LICENSE` |
| Fancybox 6.1.14 | 官方条款禁止用于 Open Source，已从本包移除 | 不要从原站再次复制；发布审计会阻止其文件或 API 残留 |
| `iowen-iconfont.woff2` 及相关图标样式 | 来源与再分发权尚未确认 | 未确认前属于公开发布阻塞项；确认授权或替换为明确开源资源 |
| `data/media/` 中的网站 Logo/产品图片 | 可能含第三方商标与美术作品 | 仅说明导航识别用途不等于取得授权；接受权利人删除请求 |
| `data/navigation-content.v1.json` 中的描述 | 事实整理与较长描述混合 | 发布者复核原创性、引用和再许可边界 |

## 当前结论

技术脱敏通过不等于授权审查通过。只要字体、主题来源或批量内容的再分发权仍不明确，就不应把整个仓库对外宣称为“全部 MIT”。可以先建私有仓库继续整理，待授权确认后再改为公开仓库。
